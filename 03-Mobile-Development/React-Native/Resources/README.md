- **"Learning React Native"** by Bonnie Eisenman
  - Building native iOS and Android apps

  - Complete project-based learning
  - Covers advanced topics
  - Production-ready code

- **"React Native Cookbook"** by Jonathan Lebensold
  - Problem-solution format
  - Practical recipes
  - Real-world scenarios
