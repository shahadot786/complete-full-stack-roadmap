### 1. The Models
- **OSI Model:** 7 Layers (Physical to Application)
- **TCP/IP Model:** 4 Layers

### 2. Addressing and Protocols
- **IP Addressing:** IPv4 vs. IPv6, CIDR blocks, Subnetting
- **TCP vs. UDP:** Reliability vs. Speed
- **Common Protocols:** HTTP/HTTPS, DNS, SSH, FTP, SMTP
- **Ports:** Well-known ports (22, 80, 443, 3306)

### 3. Troubleshooting Tools
- `ping`: Test connectivity
- `traceroute`/`tracert`: Path to a host
- `nslookup`/`dig`: DNS queries
- `netstat`/`ss`: Active connections
- `curl`/`wget`: Testing web endpoints
