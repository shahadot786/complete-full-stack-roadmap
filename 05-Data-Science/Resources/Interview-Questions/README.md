Prepare for your data science interviews.

## Statistics & Probability
1. Explain the difference between Type I and Type II errors.
2. What is the Central Limit Theorem?
3. How would you explain p-value to a non-technical person?
4. What is the difference between correlation and causation?

## SQL
1. What is the difference between WHERE and HAVING?
2. Explain the different types of JOINs.
3. Write a query to find the second highest salary.
4. What are window functions? Give an example.

## Python & Pandas
1. How do you handle missing data?
2. What is the difference between merge and join?
3. Explain groupby and agg functions.
4. How would you optimize a slow Pandas operation?

## Case Studies
1. How would you design an A/B test for a new feature?
2. Walk me through how you would analyze customer churn.
3. How would you build a recommendation system?
